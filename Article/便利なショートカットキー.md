# ショートカットキー
```
新規タグの作成：command + t
タブの移動：command + → or ←
タブを閉じる：command + w
タブの左右分割：commnad + d
タブの上下分割：commnad + shift + d
分割したタブの移動：commnad + [ or ]
検索：command + f
検索にHitした文字をたどる：Command + G（Command + Shift + G）
検索にHitした単語から選択範囲を広げる：tab（Shift + Tab）
```

```
shift ＋↓　
→選択行を増やせる

Shift　＋　option ＋　カーソル移動
→選択範囲をマウスで選択できる

Option ＋　command ＋　矢印キー
マルチカーソルにできる

Option　＋　クリック
カーソルの数を増やすことができる

Shift　＋ ⌘ ＋ L
同じ単語を一斉選択

Shift　＋ option ＋　↓
行を複製できる

Option ＋ ↓
行を移動できる

shift　＋ option ＋　F
フォーマットを修正

Option ＋ command ＋ [
→コードをおりたためる
```
