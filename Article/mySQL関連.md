# MySQLコマンド集（稼働状況・起動・再起動・停止）
```
# mysqlの稼働状況を確認
$ mysql.server status

# mysqlを停止
$ mysql.server stop

# mysqlを起動
$ mysql.server start

# mysqlを再起動
$ mysql.server restart
```
