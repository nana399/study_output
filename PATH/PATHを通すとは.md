## PATHを通すとは

- コマンド検索パスを追加すること

## コマンド検索パスとは

- シェルがコマンド実行ファイルを探しに行くパスのこと。

## コマンド検索パスの確認方法

- コマンド検索パスは環境変数$PATHに設定されており、`echo $PATH`で確認できる。

## コマンド実行ファイルの格納場所の確認方法

- `which`を利用すれば指定したコマンドの実行ファイルがどこに格納されているのか確認できる。

```
which ls
/bin/ls #実行結果
```
